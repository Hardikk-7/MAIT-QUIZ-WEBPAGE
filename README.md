# 🎓 MAIT Knowledge Quiz Web App

An interactive and responsive **web-based quiz application** developed using **HTML, CSS, and JavaScript**.
The quiz tests users on their knowledge of **Maharaja Agrasen Institute of Technology (MAIT)**, its courses, and core technologies.

This project was developed as part of the **Web Technologies (CIE-356T)** assignment.

---

## 🚀 Features

✅ Interactive quiz interface
✅ Instant feedback after each question
✅ Score tracking system
✅ Progress indicator
✅ Question randomization
✅ Responsive design (mobile-friendly)
✅ CSS animations for enhanced UX
✅ Accessible and keyboard-friendly UI
✅ MAIT-themed background design

---

## 🖥️ Technologies Used

* **HTML5** → Page structure & semantic layout
* **CSS3** → Styling, responsiveness & animations
* **JavaScript (ES6)** → Quiz logic & interactivity

---

## 📚 Quiz Functionality

* Displays one question at a time
* Highlights selected options
* Provides instant feedback:

  * ✅ Correct Answer
  * ❌ Incorrect Answer
* Tracks user score
* Shows final result at completion
* Allows quiz retake
